# 1337ify

## What is 1337ify?
1337ify is an ongoing project to translate English plaintext into leetspeak

## How to run
Clone the repository
```bash
git clone https://github.com/estoneman/1337ify.git
```
### C - 
```bash
make && ./translate <word to be translated>
```
### Python - 
Shell way
```bash
chmod +x write.py
./write.py
```
Python way
```bash
python write.py
```
